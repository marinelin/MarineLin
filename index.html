For this group project, we used Apache Spark on AWS EMR notebooks to build data processing ETL and distributed machine learning models.

Detect fake job posting

Fraudulent job postings are everywhere and they exist for various reasons. It is useful to be able to identify such job posts on a large scale.
We created a random forest baseline model and used F1 score as our north star metrics. We also created various data visualization to showcase the fake job posting by location, topic and employment type.
Tools used: Topic modeling, data visualization
